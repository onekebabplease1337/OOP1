# OOP1
Final Project Description — Swamp Defense

1. Описание проекта:
Swamp Defense - это 2D-игра формата defense-shooter, разработанная в рамках итогового проекта.
Игрок защищает объект от наступающих волн врагов, используя базовое оружие и улучшения.

2. Технологический стек:
  1) Java 17+;
  2) JavaFX (FXML, CSS);
  3) Scene Builder;
  4) SQLite (через JDBC);
  5) Java Collections.
   
3. Основная функциональность:

  1) перемещение и атака врагов по заданной траектории;
  2) несколько типов врагов (различные параметры: здоровье, скорость, урон);
  3) система волн с постепенным увеличением сложности;
  4) базовое оружие, перезарядка, метательные предметы;
  5) начисление монет и улучшение оружия;
  6) реализация нескольких экранов:
    1. Главное меню;
    2. Выбор сложности;
    3. Игровой экран;
    4. Магазин;
    5. Таблица рекордов;
    6. экран завершения игры.
  7) UI и дизайн:
  8) интерфейс построен на FXML;
  9) стилизация через CSS;
  10) применение базовых анимаций JavaFX;
  11) единый визуальный стиль приложения.
    
5. Архитектура проекта:
Код структурирован по пакетам:
  1) controllers — логика экранов;
  2) models — игровые сущности;  
  3) database — подключение к базе данных и DAO;
  4) utils — вспомогательные классы;
  5) views — FXML;
  6) assets — графика и звуки;

6. Работа с базой данных:

Используется SQLite для хранения данных игрока, результатов и улучшений.
Все SQL-файлы включены в проект.

7. Репозиторий GitHub:
Репозиторий включает:
/src — весь исходный код

/resources — изображения, звуки, базы данных
   /db — SQL файлы для создания таблиц

README.md — описание проекта, инструкция запуска

.gitignore — исключения для IntelliJ/VSCode
Репозиторий содержит исходный код, ресурсы, SQL-файлы, .gitignore и инструкцию по запуску.
Проект запускается на любом компьютере с установленным JDK 17+ и JavaFX.
